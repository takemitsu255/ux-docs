# HTMLファイル内の検索

## MAC限定
初期インストール
```
brew install ripgrep
```

## シェルの権限変更
```
chmod -R 775 ./sjis_serch.sh
```

__実行例__

<p>第一引数： 対象パス</p>
<p>第二引数：キーワード１（|でキーワード１内をor検索）</p>
<p>第三引数：キーワード１の結果を更に検索（|でキーワード2内をor検索）</p>
<p>第四引数：結果出力ファイル</p>

```
./sjis_serch.sh 対象パス 'キーワード1|キーワード2|キーワード3' 'キーワード1|キーワード2|キーワード3' result.txt
```
